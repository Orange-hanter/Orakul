# Анализ рынка платёжных киперов — Беларусь / Брест

**Дата:** май 2026  
**Фокус:** Электронные кошельки и платёжные системы, активно используемые в Республике Беларусь, с акцентом на Брестский регион.

---

## 1. Терминология

**"Кипер"** (от англ. *keeper*) — изначально обозначает приложение **WebMoney Keeper**, клиент платёжной системы WebMoney. В широком обиходе белорусские пользователи называют "кипером" любой электронный кошелёк.

---

## 2. WebMoney Keeper — профиль на рынке РБ

| Параметр | Данные |
|---|---|
| Эмитент WMB (BYN) | ОАО "Технобанк" |
| Рабочий тип кошелька в РБ | **WMB** (белорусский рубль) |
| WMZ / WME | создать можно, пополнить/использовать на территории РБ — нельзя |
| Глобальная база | ~45 млн аккаунтов, ~300 тыс. активных/нед. (данные 2020) |
| Приложения | Windows, Android, iOS |

**Особенности для Бреста:**
- Брест — приграничный город (граница с Польшей, Украиной). Исторически WebMoney WMZ/WME использовались для международных расчётов в приграничной торговле и фрилансе.
- После санкционного давления 2022+ использование нерублёвых кошельков резко снизилось.
- WMB через Технобанк остаётся рабочим инструментом для внутренних расчётов.

---

## 3. Главная инфраструктура — ЕРИП ("Расчёт")

ЕРИП — не отдельный кипер, но **основа всей платёжной экосистемы Беларуси**:

- Единое расчётно-информационное пространство, созданное НБРБ
- Интегрировано во все белорусские банки, почту, кассы
- Доступно через интернет-банкинг, мобильные приложения, терминалы
- Охватывает 60 000+ видов платежей
- В Бресте: используется повсеместно — ЖКХ, налоги, штрафы, интернет, мобильная связь

| Банк | Приложение | Охват |

## 6. Брест — региональная специфика

1. **Повышенный спрос на международные переводы** — фрилансеры, ИП в ИТ, торговля с Польшей и ЕС. Актуальны: Payoneer, Wise, Capitalist, Volet (Advcash).
2. **Бизнес-активность СЭЗ "Брест"** — резиденты используют корпоративные системы, EasyPay для эквайринга.
3. **Розничная торговля** — ЕРИП + бесконтактная оплата картами BelCard/Visa/Mastercard.
4. **Теневые и серые расчёты** — исторически активен обмен WebMoney WMZ, после 2022 г. объём существенно сократился.
5. **Криптовалюта** — в РБ легализована для физлиц через ПВТ; в Бресте есть криптообменники.

---

## 7. Тренды рынка 2024–2026

- **Упадок WebMoney** как массового инструмента — молодёжь предпочитает банковские приложения.
- **Рост переводов по номеру телефона** через ЕРИП.
- **EasyPay полностью ушёл в B2B**; как кошелёк для физлиц уже не актуален.
- **ОПЛАТИ и iPay** живут в нишах (отдельные интернет-магазины).
- **Международные альтернативы** (Payoneer, Wise) занимают нишу, освобождённую от QIWI/PayPal после санкций 2022 г.
