# Sprint 2026-05-22 — Audit-driven Stabilization

## Цель спринта

Закрыть критические риски, выявленные в полном аудите 2026-05-22:
- зафиксировать в git накопленную работу (потеря работы — топ-1 риск);
- доказать работоспособность 11 новых фич хотя бы локальным e2e;
- довести их до прод-сервера 157.22.174.219;
- закрыть открытые security-дыры (TLS, открытый порт);
- устранить задокументированные limitations (Telegram digest без venue, race condition).

## Длительность

5 рабочих дней: **2026-05-22 → 2026-05-28**.

## Метрики успеха (gate)

- [ ] Все 12 задач спринта в статусе `done` или явно отложены с обоснованием.
- [ ] Прод-сервер запущен на актуальной кодовой базе.
- [ ] HTTPS работает, `:3001` недоступен снаружи nginx.
- [ ] Пилотный клиент может пройти полный demo-сценарий из [08-pilot-demo-script.md](08-pilot-demo-script.md) без сбоев.
