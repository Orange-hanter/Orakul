## Повестка встречи

1. Видение продукта (10 мин) — Владелец продукта
2. Архитектура и стек (15 мин) — Tech Lead / Архитектор
3. Sprint Plan и DoR/DoD (15 мин) — PMO
4. Регламенты:
   - Ритуалы (daily, planning, review, retro)
   - Коммуникации (каналы, эскалации)
   - Гейты релизов
5. Риски и контакты для разблокировки (10 мин)
6. Q&A (15 мин)
