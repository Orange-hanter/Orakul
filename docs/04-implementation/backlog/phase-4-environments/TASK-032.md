## Описание

Юридические соглашения с пилотными клиентами:
- **NDA** для коммерческой тайны (обе стороны)
- **DPA** для обработки данных клиента (мы — обработчик)

Без этих документов работать с продуктовыми данными клиента незаконно.

## Критерии приёмки

- [ ] NDA подписан с каждым пилотным клиентом
- [ ] DPA подписан с каждым пилотным клиентом
- [ ] Зафиксирован объём обрабатываемых данных и срок хранения
- [ ] Записи в [Stakeholder Register](../../../05-governance/02-stakeholder-register.md)
- [ ] Документы переданы в защищённое хранилище

## Артефакт

Подписанные документы (хранение — у юриста и в защищённом хранилище).
