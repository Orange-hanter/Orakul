## Описание

Развернуть рабочее dev-окружение: репозиторий, CI/CD, БД, сервисы-зависимости, секрет-хранилище, dev-кластер. После выполнения команда может писать код и деплоить «hello world» от пуша до запуска.

## Критерии приёмки

- [ ] Репозиторий создан, доступы выданы команде по [RACI](../../02-raci-matrix.md)
- [ ] CI/CD работает: пуш → сборка → unit-тесты → деплой в dev
- [ ] БД (PostgreSQL или иная по [SAD](../phase-1-tech-docs/TASK-011.md)) развёрнута
- [ ] Сервисы-зависимости (брокер, кэш) подняты
- [ ] Секрет-хранилище настроено (Vault / cloud-secrets) и интегрировано с CI
- [ ] «Hello world» успешно прошёл pipeline и доступен в dev-окружении
- [ ] Документация по доступам и работе с окружением создана

## Артефакт

Работающее dev-окружение + краткий README в репозитории по работе с ним.
