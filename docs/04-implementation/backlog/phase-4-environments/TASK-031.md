## Описание

Подписать юридическое соглашение с Quick Resto на коммерческий доступ к API, обработку данных, SLA, ответственность сторон. Без этого договора нельзя использовать данные клиентов через API в проде.

## Критерии приёмки

- [ ] Договор подписан обеими сторонами
- [ ] DPA (Data Processing Agreement) приложен
- [ ] SLA от Quick Resto закреплён (uptime, latency)
- [ ] Прописана модель распределения ответственности (RACI на уровне договора)
- [ ] Зафиксированы условия выхода и переноса данных
- [ ] Запись в [Vendor & Partner Management](../../06-vendor-and-partner-management.md)

## Артефакт

Подписанный договор (хранение — у юриста и в защищённом репозитории).
