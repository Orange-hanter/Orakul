## Описание

Импортировать задачи из WBS в трекер, настроить эпики по модулям M1–M4, ссылки на User Stories, лейблы по [Task Standard](../../templates/task-template.md). Этот трекер становится исполнительной средой команды разработки.

## Критерии приёмки

- [ ] Трекер выбран (Jira / Linear / эквивалент) и развёрнут (или подключён существующий)
- [ ] Структура эпиков соответствует модулям PRD (M1, M2, M3, M4)
- [ ] Все задачи из WBS импортированы в трекер
- [ ] Настроены лейблы: тип, приоритет, размер по [Task Standard](../../templates/task-template.md)
- [ ] Настроена связь Epic ↔ Story ↔ Task
- [ ] Команда добавлена с правами на запись и комментирование
- [ ] Интеграция трекера с репозиторием (PR → ticket)
