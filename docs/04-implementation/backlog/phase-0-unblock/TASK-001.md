## Описание

Найти технического контактного лица в Quick Resto, договориться об условиях доступа к API, получить контакты для технических вопросов и эскалаций.

## Критерии приёмки

- [ ] Получены контакты технической поддержки Quick Resto (email, телефон, мессенджер)
- [ ] Подтверждена возможность предоставления тестового API-ключа
- [ ] Согласованы условия доступа (платный / бесплатный sandbox, ограничения)
- [ ] Контакты зафиксированы в [Stakeholder Register](../../../05-governance/02-stakeholder-register.md)

## Артефакт

Запись в [Stakeholder Register](../../../05-governance/02-stakeholder-register.md). Контакты — в защищённом хранилище проекта.
