## Описание

Детальный план пилота: расписание week-by-week, состав команды внедрения, scope фич, обучение пользователей, точки контроля, коммуникации с клиентом.

## Структура документа

1. Цели пилота и связь с KPI
2. Состав команды внедрения (ФИО, роли)
3. Scope пилота (какие модули включены)
4. Расписание по неделям:
   - Неделя 1: kick-off, аудит данных, обучение
   - Неделя 2: подключение интеграции, smoke-test
   - Неделя 3–4: эксплуатация, ежедневные стендапы
   - Неделя 5: mid-pilot review
   - Неделя 6: финал, sign-off
5. Контрольные точки (weekly check-in, mid-pilot review)
6. План коммуникаций с клиентом
7. Метрики, собираемые в ходе пилота
8. Связь с [Change Management Plan](../../03-change-management-plan.md)

## Критерии приёмки

- [ ] План содержит расписание по неделям
- [ ] Указан состав команды внедрения с ФИО
- [ ] Определены контрольные точки и формат
- [ ] План согласован с собственником пилотной точки
- [ ] Документ в статусе `✅ Утверждён`
