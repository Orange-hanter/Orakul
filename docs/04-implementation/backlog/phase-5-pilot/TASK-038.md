## Описание

Материалы для обучения пользователей пилотного клиента: краткое руководство, видео, чек-листы по ролям из [PRD §4](../../../03-product/01-product-requirements-document.md#4-целевые-пользователи-персоны).

## Структура материалов

По каждой персоне (управляющий, закупщик, шеф-повар, финконтролёр, опердир):
- Краткое руководство (что я могу делать)
- Чек-лист первых шагов (что сделать в первый день)
- Видео ≤5 мин (запись экрана с озвучкой)
- FAQ (топ-10 вопросов)

Общие материалы:
- Введение в продукт (для всех)
- Связь Orakul ↔ Kiepper
- Правила обращения с рекомендациями

## Критерии приёмки

- [ ] Подготовлено руководство для каждой персоны
- [ ] Записано ≥1 обучающего видео по каждой персоне
- [ ] Подготовлен чек-лист первых шагов
- [ ] Материалы прошли валидацию на представителях пилота
- [ ] Документы размещены в `docs/04-implementation/onboarding/`

## Артефакт

Папка `docs/04-implementation/onboarding/` со структурой по персонам + видео в общем хранилище (ссылки в документах).
