# Pricing & Commercial Model

## 0. Принципы ценообразования

1. **Value-based, не cost-plus.** Цена привязана к экономии клиента, а не к нашим затратам. Стандарт: подписка ≤10% от годовой экономии клиента.
2. **Per location, не per user.** Цена считается за точку, не за пользователя. Это устраняет трение «не давать доступ, чтобы сэкономить» и убивает adoption.
3. **Один уровень комплектации = один тариф.** Никаких 14 чекбоксов «add-on». Усложнение убивает скорость продажи.
4. **Прозрачность.** Цены публичны для тарифов Starter/Growth, договорные — только для Network.
5. **Не торгуемся скидками сверх каталога.** Любая нестандартная скидка = CR в Decision Log.

## 1. Тарифная сетка (актуальна на 2026)

| Параметр | Starter | Growth | Network |
|----------|---------|--------|---------|
| **Целевой клиент** | 3–5 точек | 6–20 точек | 21+ точек |
| **Цена за точку/мес** | 8 000 ₽ | 12 000 ₽ | от 18 000 ₽ (договорная) |
| **Минимум точек** | 3 | 5 | 10 |
| **Контракт** | 12 мес | 12 мес | 12–36 мес |
| **One-time внедрение** | 80 000 ₽ | 150–250 000 ₽ | от 400 000 ₽ |
| **Обучение** | Включено (видео + 2 ч сессии) | Включено (видео + 4 ч сессии) | Кастом |
| **CSM** | Group CSM | Назначенный CSM | Выделенный CSM |
| **SLA Uptime** | 99.0% | 99.5% | 99.7% |
| **Обновление прогноза** | 1 раз / 30 мин | 1 раз / 15 мин | 1 раз / 5 мин |
| **Глубина истории** | 12 мес | 24 мес | До 36 мес + кастом |
| **API доступ** | Read-only | Read + Write (черновики) | Полный + кастом |
| **Кастом-отчёты** | Стандартные | До 3 / квартал | Без ограничений |

## 3. Тариф Pilot (специальный)

> Главный инструмент захода на клиента. См. [GTM §6](../01-strategy/03-go-to-market.md).

| Параметр | Pilot |
|----------|-------|
| Цена | 50% от Growth (6 000 ₽/точка/мес) |
| Альтернатива | Бесплатно при условии публичного кейса |
| Срок | 6 недель |
| Точек | 1–2 |
| Внедрение | Включено |
| Гарантия | Возврат оплаты, если KPI пилота не достигнуты по верифицированным данным |
| Обязательства клиента | Куратор, доступ к данным Kiepper, 2 интервью/нед, согласие на кейс |
| Конверсия в Paid | Целевая ≥80% |

> Pilot не продлевается дольше 6 нед. Либо переход на Paid, либо завершение и разбор причин — это репутационная защита.

## 4. Скидки и стандартные модификаторы

| Тип | Размер | Условие |
|------|--------|---------|
| **Annual prepay** | −10% | Полная оплата за 12 мес вперёд |
| **Multi-year** | −15% | Контракт на 24 мес |
| **Network volume** | −15% | 10+ точек (только Growth) |
| **Case partnership** | −10% | Согласие на публичный кейс с цифрами |
| **Pre-IPO / стратегический клиент** | −20% | По решению CEO, ≤2 в год |
| **Pilot → Paid** | −5% на год 1 | Бесшовный переход в течение 14 дней |

> **Stacking запрещён сверх двух скидок одновременно.** Кумулятивная скидка >30% — только через CR.

## 5. Дополнительные платные опции

| Опция | Цена | Условие |
|-------|------|---------|
| Кастомная интеграция (не Kiepper-стандарт) | T&M (от 250 000 ₽) | Network only |
| Дополнительный CSM-канал (выделенный менеджер) | +30% к подписке | Опция |
| Ускоренный onboarding (≤14 дней) | +50% к стоимости внедрения | Опция |
| Расширенный SLA (99.9%) | +20% к подписке | Network only |
| Локальный/on-premise хостинг | T&M, мин. 2 млн ₽ | Большие сети, требования безопасности |

## 6. Условия контракта (стандарт)

| Параметр | Значение |
|----------|----------|
| Срок контракта | 12 мес (минимум) |
| Auto-renewal | Да, опт-аут за 30 дней до окончания |
| Платёж | Ежемесячно или предоплата 12 мес |
| NDA | До переговоров; можно подписать наш стандартный |
| DPA (Data Processing Agreement) | Обязательно перед стартом E1 |
| Изменение цены | Ежегодное, не более CPI + 5%, с уведомлением за 60 дней |
| Pricing-lock | 24 мес для клиентов первой волны |
| Расторжение клиентом | За 30 дней без штрафа в первые 90 дней; далее — по условиям контракта |
| Расторжение нами | За 60 дней с обоснованием (только при нарушениях) |
| Возврат данных | По требованию клиента в стандартных форматах в течение 30 дней |
| SLA-кредиты | Если uptime ниже плана: компенсация ≤20% месячного платежа |

## 8. Защита от ценовой эрозии

| Угроза | Защита |
|--------|--------|
| «Дайте скидку, как у клиента X» | Скидки только в каталоге; индивидуальные исключения = CR |
| «Конкурент дешевле» | Сравниваем не цены, а валовую ценность (annual savings); ROI-калькулятор |
| Просьба о бесплатной услуге сверх контракта | Любая работа сверх scope — отдельный T&M или PR (Project Request) |
| Запрос «отключить часть функций — снизить цену» | Тарифы фиксированы; «down-tier» возможен только в окно продления |
